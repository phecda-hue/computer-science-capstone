# 결정 엔진 명세 v2.0

## 위험도 공식
Risk = W × (1/d² + v_approach/d)

| 변수        | 설명                                      |
|-------------|-------------------------------------------|
| W           | 클래스 가중치 (아래 표 참조)              |
| d           | IIR 필터링된 깊이 (미터)                  |
| v_approach  | max(0, v_relative) — 접근 속도만 사용     |

## 클래스 가중치 W
| class_id | 클래스      | W   |
|----------|-------------|-----|
| 0        | stairs      | 0.8 |
| 1        | curb        | 0.6 |
| 2        | manhole     | 0.5 |
| 3        | person      | 0.5 |
| 4        | hole        | 0.9 |

## 위험 등급
| 등급   | risk_score  | 알림               |
|--------|-------------|------------------  |
| HIGH   | ≥ 2.0       | 즉시 음성 + 진동   |
| MEDIUM | 0.8 ~ 2.0   | 경고 음성          |
| LOW    | 0.3 ~ 0.8   | 안내 음성          |
| SAFE   | < 0.3       | 없음               |

## depth 추정 실패 시 fallback
depth_m = -1.0이면 YOLO bbox 면적으로 거리 추정
bbox_area가 없으면 위험도 = 0 (SAFE)

## 필터링 규칙
- ROI: 이미지 하단 50% 범위만 사용
- 노이즈: 주변 픽셀 중앙값과 0.5m 이상 차이나는 픽셀 제거
- 최소 픽셀: 유효 픽셀 10개 미만이면 장애물 미인식
- IIR: y_n = 0.3 × x_n + 0.7 × y_{n-1}
